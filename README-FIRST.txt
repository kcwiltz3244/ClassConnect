CLASSCONNECT VERSION 2.4 — WELCOME HOME

UPLOAD OR REPLACE:
- index.html
- styles.css
- app.js
- manifest.webmanifest
- service-worker.js
- classconnect-logo.png
- icon-192.png
- icon-512.png

DO NOT REPLACE:
- config.js
- admin.html
- admin.js
- any backup files

UPLOAD STEPS:
1. Open the ClassConnect GitHub repository.
2. Choose Add file > Upload files.
3. Upload the eight files listed above.
4. Commit the changes.
5. Wait for GitHub Pages to deploy.
6. Refresh the website.

PHONE ICON:
If the old icon remains on an existing home-screen shortcut, remove that shortcut and add ClassConnect to the home screen again.
