CLASSCONNECT 2.1 STREAMLINED

This version keeps the Living Yearbook features but is designed to handle 100+ classmates.

IMPROVEMENTS
- Compact Hallway quick-launch buttons
- Compact list view as the default
- Optional photo-grid view
- 20 classmates per page
- Expandable details and photos
- Faster browsing and less scrolling
- Who's New limited to three classmates

INSTALL
1. Keep the same Code.gs and Google Sheet structure from ClassConnect 2.1.
2. Open config.js and paste your existing Apps Script /exec URL.
3. Upload these files to GitHub:
   index.html
   styles.css
   app.js
   config.js
   manifest.webmanifest
   service-worker.js
   icon.svg
4. Commit the changes.
5. Netlify will redeploy automatically.

This is a front-end replacement only. You do not need to change Apps Script again.
